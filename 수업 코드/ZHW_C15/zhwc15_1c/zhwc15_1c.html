*& [HW1-C] 구구단 계산하기 C15

REPORT ZHWCC15_1C.

** 변수이름이 8자로 제한이 있음 pa_12345678 안됨, 실무에서는 P_123456 이런식으로 한글자가 아까워서 아낌
*PARAMETERS PA_NUM TYPE I. " 입력 받은 구구단의 단수 "
*
*IF PA_NUM < 2 OR PA_NUM > 9.
*  MESSAGE ' 입력한 단수가 2보다 작거나 9보다 큽니다. ' TYPE 'E'. " 구구단의 범위가 2 - 9 단을 벗어날 경우 오류 메시지 출력 "
*ENDIF.
*
*DATA: GV_RESULT TYPE I. " 곱셈 결과를 저장하는 변수 "
*
** 구구단 실행
*DO 9 TIMES. " 입력받은 단의 9곱 까지 계산 "
*  GV_RESULT = PA_NUM * SY-INDEX. " 입력 받은 단의 현재(SY-INDEX) 곱  "
*  WRITE: / PA_NUM, ' * ', SY-INDEX, ' = ', GV_RESULT.
*ENDDO.

PARAMETERS PA_DAN TYPE I. " 입력 받은 구구단의 단수 "

IF PA_DAN < 2 OR PA_DAN > 9.
* TEXT-M01 해도됨
  MESSAGE ' 입력한 단수가 2보다 작거나 9보다 큽니다. ' TYPE 'S' DISPLAY LIKE 'I'. " 구구단의 범위가 2 - 9 단을 벗어날 경우 오류 메시지 출력 "
  EXIT. " 조건이 많아지면 햇갈리니 EXIT을 사용한다. IF 문에서 사용하면 프로그램 EXIT, LOOP 에서는 LOOP 를 나감

ENDIF.

* 사용자가 보기 이쁘게 표현하려고 이렇게 출력이 이쁘게 나옴
DATA GV_MUL TYPE N LENGTH 1.
DATA GV_REUSLT TYPE N LENGTH 2.

DO 9 TIMES.
  GV_MUL = SY-INDEX.
  GV_REUSLT = PA_DAN * GV_MUL.

  WRITE / PA_DAN.
  WRITE 'X'.
  WRITE GV_MUL.
  WRITE '='.
  WRITE GV_REUSLT.

ENDDO.
