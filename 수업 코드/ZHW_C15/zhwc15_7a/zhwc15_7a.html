*&---------------------------------------------------------------------*
* 승객이 버스에 탑승 하기전에 HANDLER 등록
*&---------------------------------------------------------------------*
  " 버스 탑승 EVENT 를 등록한다.
  " 어떤 사람이든 BUS 에 탑승하면 작동을 해야함으로 FOR 각 승객 OBJ 를 사용하지 않고
  "  FOR ALL INSTANCES 를 사용한다.
  "  왜냐면 빈 버스가 도착해도 RAIS ARRIVE_TERMINAL 로인해
  " ON_GET_ON_BUS 가 작동하여 승객이 하차한 것으로 기록을 해버린다.
  SET HANDLER LCL_EVENT_HANDLER=>ON_GET_ON_BUS FOR ALL INSTANCES.
*&---------------------------------------------------------------------*


*&---------------------------------------------------------------------*
* 승객이 버스에 탑승

* raise error 로 경고 메세지가 나옴
*  PERFORM GET_ON_BUS USING '이의정'
*                         '마을버스 01'.

  PERFORM GET_ON_BUS USING '박설아'
                         '마을버스 01'.

  PERFORM GET_ON_BUS USING '박현진'
                       '광역버스 8800'.


* 버스의 수익 총합 출력

* 바로 앞에 버스가 내려서 MO_BUS 가 비어 있는  바로 직전 승객에게 강제 종점하차
* 경고 메세지가 안나오고 handler 가 해제가 되었음으로 머스가 종점에 도착했다는 메세지만 나옴
  PERFORM ARRIVE_TERMINAL USING '광역버스 8800'.
