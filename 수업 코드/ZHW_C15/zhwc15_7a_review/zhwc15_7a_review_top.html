Code listing for: ZHWC15_7A_REVIEW_TOP

Description: TOP
*&---------------------------------------------------------------------*
*& Include          ZHWC15_7A_REVIEW_TOP
*&---------------------------------------------------------------------*

* 아직 CLASS 정의를 하지 못한 상황에서 바로 CLASS 를 사용하려 하면
* 미정의된 TYPE 으로 오류가 발생한다.
* 1. 그럼으로 TOP 을 선언하기 전에 CLASS 를 정의해야 한다.( INCLUDE CLS가 먼저 와야함 )
* 2. 미리 CLASS 의 존재를 미리 알려준다 ( DEFERRED ).

* 나중에 CLASS 를 만들게 오류지 말고 일단 기다려줘
CLASS LCL_EVENT_HANDLER DEFINITION DEFERRED.

CLASS LCL_PERSON DEFINITION DEFERRED. " DEFERRED 를 사용하면 향후 무조건 CLASS 를 정의 해줘야 오류가 안남
DATA: GO_PERSON TYPE REF TO LCL_PERSON.

CLASS LCL_BUS DEFINITION DEFERRED.
DATA: GO_BUS TYPE REF TO LCL_BUS.

* 내가 원하는 이름을 가진 OBJ 를 찾기 위해서 ITAB을 만듬
" PERSON 를 모아 두는 ITAB.
DATA: BEGIN OF GS_PERSON,
        NAME   TYPE FULLNAME,
        PERSON TYPE REF TO LCL_PERSON,
      END OF GS_PERSON,
      GT_PERSON LIKE TABLE OF GS_PERSON.

" BUS 를 모아 두는 ITAB.
DATA: BEGIN OF GS_BUS,
        NUMBER TYPE CHAR20,
        BUS    TYPE REF TO LCL_BUS,
      END OF GS_BUS.

DATA: GT_BUS LIKE TABLE OF GS_BUS.
