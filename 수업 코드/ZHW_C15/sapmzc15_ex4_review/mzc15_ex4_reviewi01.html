MODULE EXIT_100 INPUT.

  CASE OK_CODE.
    WHEN 'EXIT'.
      LEAVE PROGRAM.
    WHEN 'CANC'.
      " SINGLE SCREEN TRANSACTION - 현재화면 초기화
      " MULTI SCREEN TRANSACTION - 이전화면 이동(BACK)

      " 현재화면 초기화는 어떻게 하는가?
      " 1. 화면의 입출력 필드와 같은 이름의 변수 클리어
      " 다음 PBO 실행시 초기화된 GS_AIRLINE 이 ZTC15_EX4_REVIEW 로 전달되어
      " ZTC15_EX4_REVIEW가 초기화 됨으로써 화면에 있는 값이 초기화됨
      CLEAR GS_AIRLINE.
      CLEAR ZTC15_EX4_REVIEW.  " 그냥다 초기화 하자 귀찮다
      " 2. SAP 메모리 초기화
      SET PARAMETER ID 'CAR' FIELD ZTC15_EX4_REVIEW-CARRID.
      " 3. 현제 화면 버리고 NEXT SCREEN 이동
      LEAVE SCREEN. " MOVE NEXT SCRREN. 다음화면으로 이동
      " 다음화면이 없으면 현재 화면으로 이동

  ENDCASE.

ENDMODULE.

MODULE USER_COMMAND_0100 INPUT.

    WHEN 'BACK'.

  ENDCASE.

ENDMODULE.
