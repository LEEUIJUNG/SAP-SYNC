Code listing for: ZHWC15_6A_SCR

Description: SCREEN
*&---------------------------------------------------------------------*
*& Include          ZHWC15_6A_SCR
*&---------------------------------------------------------------------*
*& SELECTION SCREEN 을 선언 하는 INDLUDE 파일이다.
*& 실행모드, 검색 조건, 검색 조건(선택 사항) 을 입력 받는다.
*&---------------------------------------------------------------------*

* -------------------------------------------------------------------------- *
* RADIO BUTTON 을 다루는 SCREEN BLOCK
* -------------------------------------------------------------------------- *
SELECTION-SCREEN BEGIN OF BLOCK BLOCK1 WITH FRAME TITLE TEXT-T01.

  SELECTION-SCREEN BEGIN OF LINE. " RADIO BTN 을 한줄에 적음

    " RADIO BTN 을 3개를 만든다.
    " 각각의 버튼은 같은 GROUP 으로 묶어서 한 요소만 선택이 가능하도록 한다.
    " RADIO BTN 클릭 이벤트를 받기 위해 USER COMMAND 를 정의한다.
    " 향후 SCREEN EVENT 를 다루기 위해 MODIF ID 를 부여한다.

    " RADIO BTN 의 시작 위치 (1)
    SELECTION-SCREEN POSITION 1.
    PARAMETERS: PA_R1 RADIOBUTTON GROUP FC1 USER-COMMAND UC1 DEFAULT 'X' .
    " RADIO BTN 에 표시될 이름을 (출력길이 20) PA_R1 에서 가져온다. (TEXT ELEMENT에 선언)
    " 기본 검색 조건을 fc1 btn (insert data) 에 표시한다.
    SELECTION-SCREEN COMMENT (20) FOR FIELD PA_R1 .
    SELECTION-SCREEN POSITION 25.
    PARAMETERS: PA_R2 RADIOBUTTON GROUP FC1.
    SELECTION-SCREEN COMMENT (20) FOR FIELD PA_R2.
    SELECTION-SCREEN POSITION 49.
    PARAMETERS: PA_R3 RADIOBUTTON GROUP FC1.
    SELECTION-SCREEN COMMENT (20) FOR FIELD PA_R3 .

  SELECTION-SCREEN END OF LINE.


SELECTION-SCREEN END OF BLOCK BLOCK1.
* -------------------------------------------------------------------------- *


* 삭제모드시 비활성화 설정을 위해 ID 을 설정한다. MODIF ID = M1
* -------------------------------------------------------------------------- *
* 검색조건을 받기 위한 SCREEN BLOCK
* -------------------------------------------------------------------------- *
SELECTION-SCREEN BEGIN OF BLOCK BLOCK2 WITH FRAME TITLE TEXT-T02.

  SELECT-OPTIONS: SO_CAR FOR GV_CARRID OBLIGATORY MODIF ID M1 , " CARRID 는 필수로 입력해야한다.
                  SO_CON FOR GV_CONNID MODIF ID M1,
                  SO_FLD FOR GV_FLDATE MODIF ID M1,
                  SO_OD FOR GV_ORDER_DATE MODIF ID M1.

SELECTION-SCREEN END OF BLOCK BLOCK2.
* -------------------------------------------------------------------------- *


* -------------------------------------------------------------------------- *
* 선택적인 추가 검색 조건을 받기 위한 SCREEN BLOCK
* -------------------------------------------------------------------------- *

SELECTION-SCREEN BEGIN OF BLOCK BLOCK3 WITH FRAME TITLE TEXT-T03 NO INTERVALS.

    SELECT-OPTIONS: SO_CTRF FOR GV_COUNTRYFR MODIF ID M1  ,
                  SO_CTYF FOR GV_CITYFROM MODIF ID M1  ,
                  SO_CTRT FOR GV_COUNTRYTO MODIF ID M1  ,
                  SO_CTYT FOR GV_CITYTO MODIF ID M1  .

    PARAMETERS: PA_CNC AS CHECKBOX MODIF ID M1. " 예약 취소정보를 선택하기 위한 CHECK BOX

SELECTION-SCREEN END OF BLOCK BLOCK3.
