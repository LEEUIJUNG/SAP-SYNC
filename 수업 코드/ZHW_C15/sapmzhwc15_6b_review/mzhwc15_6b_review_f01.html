*& Form SELECT_COUNTRY_NAME
*&---------------------------------------------------------------------*
*& text
*&---------------------------------------------------------------------*
*&      --> SPFLI_COUNTRYFR
*&      <-- SCR_0120_LANDX_FROM
*&---------------------------------------------------------------------*
* 국가 코드만 시용, 국가 명만 반환.
FORM SELECT_COUNTRY_NAME USING VALUE(PV_COUNTRY) TYPE T005T-LAND1
                          CHANGING VALUE(CV_LANDX) TYPE T005T-LANDX.

  " 만약 국가명 코드가 없으면 DB TAB T005T 에서 DATA를 가져온다.
  IF GT_T005T IS INITIAL.
    SELECT LAND1 " 국가코드
           LANDX " 국가명
      FROM T005T
      INTO CORRESPONDING FIELDS OF TABLE GT_T005T.

    " 국가 코드 기준으로 정렬
    SORT GT_T005T BY LAND1.

  ENDIF.

  " 모든 국가명을 보유한 ITAB GT_T005T 를 대상으로 조회를 진행
  READ TABLE GT_T005T INTO GS_T005T WITH KEY LAND1 = PV_COUNTRY BINARY SEARCH.

  IF SY-SUBRC = 0.
    " 결과가 있으면 값을 CV_LANDX 로 전달
    CV_LANDX = GS_T005T-LANDX.
  ELSE.
    " 값이 없다면 빈값 전달
    CLEAR CV_LANDX.
  ENDIF.

ENDFORM.
