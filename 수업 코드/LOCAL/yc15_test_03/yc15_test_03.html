REPORT YC15_TEST_03.

PARAMETERS pa_msg TYPE string.
PARAMETERS pa_type TYPE c.

CASE pa_type.
  WHEN 'I'.
    MESSAGE '정보성 메세지 입니다.' TYPE pa_type.
  WHEN 'S'.
    MESSAGE '성공 메세지 입니다.' TYPE pa_type.
  WHEN 'E'.
    MESSAGE '에러 메세지 입니다.' TYPE pa_type.
    MESSAGE '검사용 메세지 입니다.' TYPE 'I'. "문구가 나오지 않아야 정상, 'E' 위 과정에서 프로그램이 종료됨"
  WHEN OTHERS.
    MESSAGE '이건 허용되지 않은 타입입니다.' TYPE 'I' DISPLAY LIKE 'E'.
    MESSAGE '실제로는 에러 메세지가 아니라서 중단되지 않습니다..' TYPE 'I'.

ENDCASE.
