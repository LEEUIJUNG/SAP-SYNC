MODULE FILL_DYNNR OUTPUT.

* Tablstrip 에서 현재 활성화 되어 있는 tap 에 해당하는 화면을 보여준다.
* 첫번째 tap 은 110, 두번째는 120, 세번째는 130 번 화면이 보여야한다.
  CASE MY_TABSTRIP-ACTIVETAB. " 현재 활성화 되어 있는 화면
    WHEN 'FC1'. " 첫번째 탭의 Function Code 는 FC1 이다

      DYNNR = '0110'.

    WHEN 'FC2'. " 두번째 탭의 Function Code 는 FC2 이다

      DYNNR = '0120'.

    WHEN 'FC3'. " 세번째 탭의 Function Code 는 FC3 이다

      DYNNR = '0130'.

    WHEN OTHERS. " 모두에 해당하지 않으면  초기값 설정

      DYNNR = '0110'. " 첫화면에 110번 서프스크린이 보이도록 취급
      MY_TABSTRIP-ACTIVETAB = 'FC1'. " 첫번째 탭을 활성화 탭으로 취급

  ENDCASE.

ENDMODULE.

MODULE GET_SPFLI OUTPUT.

* 항공편 정보를 키필드 기준으로 조회한다.
* 사용자가 비행일정을 조회하기 위해 입력했던 검색조건 중 항공사 ID와 항공편번호는
* SPFLI 테이블의 키필드에 해당되는 정보라서 단 하나의 데이터를 검색할 수 있는 조건이 된다.

* 검색된 데이터는 화면에 출력 시키기 위해 SDYN_CONN에 적절하게 값을 전달한다.


  SELECT SINGLE * FROM SPFLI
    INTO CORRESPONDING FIELDS OF SDYN_CONN
    WHERE CARRID = SDYN_CONN-CARRID
    AND CONNID = SDYN_conn-CONNID.

ENDMODULE.
