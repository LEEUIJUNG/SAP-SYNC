LOAD-OF-PROGRAM.

* 1. SAP Memory 에서 기록된 값을 가져온다.
* 2. 기록된 값을 기준으로 Database Table Sflight 를 조회
* 3. 조회한 결과는 GS_FLIGHT에게 전달

  " SAP 에 기록된 값을 가져오는 KEY WORD
  " GET PARAMETER ID ( ' 파라미터 ID 값 ', ' ' 따음표를 적어줘야함 문자열이여야함 ) FIELD ( 메모리에 기록된 값을 전달 받을 변수 )
  GET PARAMETER ID: 'CAR' FIELD GS_FLIGHT-CARRID,
                    'CON' FIELD GS_FLIGHT-CONNID,
                    'DAY' FIELD GS_FLIGHT-FLDATE.

  SELECT SINGLE *
    FROM SFLIGHT
    INTO GS_FLIGHT
    WHERE CARRID = GS_FLIGHT-CARRID
    AND CONNID = GS_FLIGHT-CONNID
    AND FLDATE =  GS_FLIGHT-FLDATE.
