MODULE MODIFY_SCREEM OUTPUT.

* Radio Button과 연결된 3개의 변수 중 비행정보 관리에 대한 선택일때만
* 아래로직이 진행되도록 Check
  CHECK MAINTAIN_FLIGHTS = 'X'.

  LOOP AT SCREEN.
    " 입력필드가 SDYN_CONN 의 PLANETYPE 인 경우 (비행정보 중 학공기 모델)
    IF SCREEN-NAME = 'SDYN_CONN-PLANETYPE'.

      " 입력이 가능하도록 input option 활성화
      SCREEN-INPUT = 1.

    ENDIF.

    MODIFY SCREEN.

  ENDLOOP.
ENDMODULE.
