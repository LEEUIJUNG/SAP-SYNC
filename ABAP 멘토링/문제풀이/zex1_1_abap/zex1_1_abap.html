*&---------------------------------------------------------------------*
*& Report ZEX1_1_ABAP_A
*&---------------------------------------------------------------------*
*& 구구단을 출력하는 ABAP Program을 작성한다.
*&
*& 5단을 입력했을 경우 2단부터 5단까지 출력한다.
*& 2단 미만을 입력하거나 9단 이상을 입력 할 경우 에러 메시지를 출력한다.
*& 2단 미만 입력 시: 2단 이상을 입력하세요.
*& 9단 초과 입력 시: 9단 이하를 입력하세요.
*&
*&---------------------------------------------------------------------*

REPORT ZEX1_1_ABAP.


* TEXT-001 = '단수'. " data element 001 단수를 생성 "

* selection 을 설정
SELECTION-SCREEN BEGIN OF LINE. " begin of line 을 사용 하면 1줄로 적용 "
  SELECTION-SCREEN COMMENT 1(10) TEXT-001 FOR FIELD PA_NUM. " pa_num을 data element text_001 로 selection 화면에 표시 "
  PARAMETERS PA_NUM TYPE I.

SELECTION-SCREEN END OF LINE.

DATA: GV_INNER_IDX TYPE I,
      GV_OUTER_IDX TYPE I,
      GV_TIMES     TYPE I.



* 문제에 주어진 조건에 따라 오류메세지 출력
IF PA_NUM < 2 .
  MESSAGE ' 2단 이상을 입력하세요. ' TYPE 'S' DISPLAY LIKE 'I'.
  EXIT.

ELSEIF PA_NUM > 9.
  MESSAGE ' 9단 이하를 입력하세요. ' TYPE 'S' DISPLAY LIKE 'I'.
  EXIT.

ELSE.
  GV_TIMES = PA_NUM - 1 .

* 구구단 출력을 진행
  DO GV_TIMES  TIMES. " 구구단이 2단부터 출력이 됨으로 입력 받은 pa_num 에 - 1 을 함

    GV_OUTER_IDX = SY-INDEX + 1. " 처음 받는 단수는 2단부터 임으로 외부 반복문의 index에 + 1 을 함

    DO 9 TIMES.
      GV_INNER_IDX = GV_OUTER_IDX * SY-INDEX. " 내부 반복문의 index를 사용하여 구구단의 곱샘을 수행
      WRITE :/ GV_OUTER_IDX , ' * ' , SY-INDEX , ' = ' , GV_INNER_IDX.

    ENDDO.

    ULINE.

  ENDDO.

ENDIF.
