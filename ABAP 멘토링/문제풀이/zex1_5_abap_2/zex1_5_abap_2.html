*--------- test15 SS-OUTPUT 적용 ---------*
* SS-OUPUT 은 사용자가 SCREEN1000 에서 동작을 하였을때 이를 반영한다.
  " 해당 RADIO BUTTON 이 눌렀을떄를 기준으로 한다
  " SCREEN의 LOOP 를 돌려 위에 지정한 MODIFY ID 를 찾는다.
  " 찾은 MODIFY ID ( MODIFY iD 는 SCREEN-GROUP1 로 지정이됨) 기준으로 SCREEN 의 COMPONENT 를 변경한다
  " INVISIBLE , ACTIVATE, INPUT 등등 이 가능하다
  " 그후 MODIFY SCREEN 을 사용하여 화면에 반영을 한다.
  IF GV_RADI1 = 'X'.
    LOOP AT SCREEN.
      IF SCREEN-GROUP1 = 'T1'.
*        SCREEN-INVISIBLE = 0. " 화면이 초기화 되면 기본 값인 0으로 다시 돌아와서 작성필요 없음
      ELSEIF SCREEN-GROUP1 = 'T2'.
        SCREEN-INVISIBLE = 1.
      ELSEIF SCREEN-GROUP1 = 'T3'.
        SCREEN-INVISIBLE = 1.
      ENDIF.
      MODIFY SCREEN.
    ENDLOOP.

  ELSEIF  GV_RADI2 = 'X'.
    LOOP AT SCREEN.
      IF SCREEN-GROUP1 = 'T1'.
        SCREEN-INVISIBLE = 1.
      ELSEIF SCREEN-GROUP1 = 'T2'.
*        SCREEN-INVISIBLE = 0. " 화면이 초기화 되면 기본 값인 0으로 다시 돌아와서 작성필요 없음
      ELSEIF SCREEN-GROUP1 = 'T3'.
        SCREEN-INVISIBLE = 1.
      ENDIF.
      MODIFY SCREEN.
    ENDLOOP.

  ELSEIF  GV_RADI3 = 'X'.
    LOOP AT SCREEN.
      IF SCREEN-GROUP1 = 'T1'.
        SCREEN-INVISIBLE = 1.
      ELSEIF SCREEN-GROUP1 = 'T2'.
        SCREEN-INVISIBLE = 1.
      ELSEIF SCREEN-GROUP1 = 'T3'.
*        SCREEN-INVISIBLE = 0. " 화면이 초기화 되면 기본 값인 0으로 다시 돌아와서 작성필요 없음
      ENDIF.
      PA_CAR1 = 'TEST'. " FIELD 에 값을 수동으로 넣을 수 있다.
      MODIFY SCREEN.
    ENDLOOP.

  ELSEIF  GV_RADI4 = 'X'.
    " SCREEN BLCOK 전체를 가리려면 모든 구성 요소들을 INVISILBE 해야한다.
    LOOP AT SCREEN.

      ENDIF.

  ENDIF.
